# Deep AI Signal Node

Dark hacker-style chat UI built with React, Vite, and Tailwind CSS.


**##  Go and choose Release Option and use Download my model deep_ai and run locally.**


## Run**

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The interface includes the animated canvas backdrop, scanning line, glitch logo treatment, chat history sidebar, code block rendering, typing indicator, and prompt-style input bar.
